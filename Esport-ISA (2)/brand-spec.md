# ISA Esport Brand Spec

## Visual Direction: Tech / Utility (Esport)
A high-contrast, green-and-white theme that feels professional and technical.

## Tokens
- `--bg`:      oklch(99% 0.005 145) /* Almost white with a hint of green */
- `--surface`: oklch(100% 0 0)      /* Pure white cards */
- `--fg`:      oklch(15% 0.02 145)  /* Deep dark green-black for text */
- `--muted`:   oklch(55% 0.02 145)  /* Muted green-grey */
- `--border`:  oklch(90% 0.02 145)  /* Subtle border */
- `--accent`:  oklch(68% 0.23 145)  /* Vibrant ISA Green */

## Typography
- **Display**: -apple-system, BlinkMacSystemFont, 'Inter', 'Segoe UI', system-ui, sans-serif
- **Body**:    -apple-system, BlinkMacSystemFont, 'Inter', 'Segoe UI', system-ui, sans-serif
- **Mono**:    'JetBrains Mono', 'IBM Plex Mono', ui-monospace, Menlo, monospace

## Posture
- Hairline borders (1px)
- Tabular numbers for scores and seeding
- Subtle grid background
- Accent used for "Winner" indicators and primary CTAs
- Smooth transitions for bracket updates
